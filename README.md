# git使用
######注意
	1. master是项目主分支(默认)；
	2. git 的3个区：a=>b=>c
		a.工作区:项目的这块
		b.暂存区:帮我们保存的
			作为过渡层，避免误操作，保护工作区和版本区，分支处理
		c.版本区(库):
	3.命令：
		git status 显示工作目录和暂存区的状态
		git add xxx 将xxx(目录或文件)修改添加到暂存区
###git使用完整步骤
####1. git clone https://github.com/zhengwei2531/drag1.git 把GitHub上的项目克隆到本地
